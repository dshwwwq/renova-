Описание проекта

Одностраничный сайт (landing page) для компании «Ренова+» (г. Уфа).
Сайт содержит информацию об услугах компании, реализованных проектах, контактах и реквизитах.
Структура сайта

    Шапка — логотип компании, навигация

    Приветственный блок — описание команды, перечень услуг (строительство, ремонт, отделка, дизайн), кнопки навигации

    Проекты — список реализованных и текущих объектов 
    Контакты — телефон, email, адрес офиса, режим работы

    Подвал — копирайт, адрес, реквизиты (ИНН, ОГРН)

Технологии

    HTML5

    CSS3 (адаптивная вёрстка, Flexbox/Grid)

    Git / GitHub

Работа с Git
